# RL-Based-Radiation-Aware-6T-SRAM-180nm
Radiation-resilient 6T SRAM in 180nm CMOS optimized using Reinforcement Learning and Dynamic Body Biasing. Sim-in-the-loop Cadence Spectre framework maximizing Qcrit and SNM under transient radiation injection.
This project presents a radiation-resilient 6T SRAM cell in 180nm CMOS optimized using Reinforcement Learning (RL) and Dynamic Body Biasing (DBB).
The objective is:
Maximize Critical Charge (Qcrit)
Maximize Static Noise Margin (SNM)
Minimize Delay
Minimize Power
Under transient radiation injection.
